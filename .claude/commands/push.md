# Git Push Command

作業完了後にGitHubにプッシュするコマンドです。

## 実行手順

1. 変更されたファイルをすべてステージング
2. コミットメッセージを生成（変更内容に基づく）
3. GitHubにプッシュ

## 注意事項

- コミットメッセージは変更内容を要約して自動生成します
- プッシュ前に必ず変更内容を確認してください

---

以下を実行してください：

1. `git status` で変更ファイルを確認
2. `git add .` ですべての変更をステージング
3. 変更内容に基づいて適切なコミットメッセージを作成
4. `git commit` でコミット
5. `git push -u origin main` でプッシュ

コミットメッセージのフォーマット：
```
[タイプ] 変更の概要

- 詳細1
- 詳細2
```

タイプ例: feat, fix, style, refactor, docs, chore
