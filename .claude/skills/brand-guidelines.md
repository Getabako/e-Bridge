# e-Bridge ブランドガイドライン

## サービス概要
**e-Bridge（イーブリッジ）**
吃音・不登校経験・発達特性のある小中高生向けの教育支援サービス

「e（education/everyone）」と「Bridge（橋渡し）」を組み合わせた名前。
子どもたちと未来をつなぐ橋渡しをする。

## ターゲットユーザー

### 一次ターゲット（本人）
- 小学生〜高校生
- 吃音のある子
- 不登校・不登校経験のある子
- 発達特性（ADHD、ASD等）のある子

### 二次ターゲット（保護者・支援者）
- 保護者
- 学校の先生
- 療育・支援機関のスタッフ

## ブランドボイス（トーン・言葉遣い）

### 心がける言葉遣い
- 温かく、寄り添う
- 押し付けがましくない
- 希望を感じさせる
- 分かりやすい言葉
- 専門用語は避ける

### 良い例
- 「あなたのペースで大丈夫」
- 「一緒に考えていきましょう」
- 「まずはお話を聞かせてください」
- 「小さな一歩から」

### 避ける表現
- 「すぐに改善！」（プレッシャー）
- 「問題解決」（問題扱い）
- 「治す」「治療」（医療用語）
- 「普通になれる」（普通の押し付け）

## アクセシビリティ要件

### 必須対応
1. コントラスト比 WCAG AA 準拠（4.5:1以上）
2. キーボード操作対応
3. スクリーンリーダー対応
4. フォーカスインジケーター明確
5. 色だけに依存しない情報伝達
6. 十分な余白とクリックターゲット

### 発達特性への配慮
1. 情報量を適切に制限
2. 予測可能な動作
3. 過度なアニメーション禁止
4. 感覚過敏への配慮（明滅・強いコントラスト避ける）
5. 明確な構造とナビゲーション

## 禁止事項

1. **絵文字の多用** - 必要最小限に
2. **紫色の使用** - AIっぽさを避ける
3. **ロボット・AI画像** - 人間味を大切に
4. **過度なアニメーション** - 感覚過敏への配慮
5. **ネガティブな表現** - 問題視しない
6. **プレッシャーを与える表現** - 焦らせない
